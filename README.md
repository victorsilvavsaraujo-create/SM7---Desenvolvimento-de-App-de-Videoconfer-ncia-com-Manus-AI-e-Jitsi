### 📝 Descrição

Este projeto é uma plataforma de videoconferência para Android que redefine a colaboração remota. Utilizando o SDK open-source do **Jitsi Meet**, a aplicação foi estruturada e otimizada com o auxílio da **Manus AI**, focando em um nicho de mercado específico: **[Insira aqui o seu nicho, ex: Mentoria para Desenvolvedores ou Consultas de Telemedicina Gamificadas]**.



O diferencial reside na camada de inteligência aplicada ao fluxo da chamada, transformando uma simples reunião em uma experiência interativa e produtiva.

Para o seu README do projeto Videoaula App, o ideal é um texto que explique de forma clara e técnica como você resolveu o problema da comunicação digital.



Aqui está o texto formatado para você copiar:



📹 Videoaula App – Conectando Educação e Tecnologia

🎯 O Problema

No cenário atual de ensino híbrido e remoto, o acesso a plataformas de videoconferência muitas vezes esbarra em interfaces complexas, necessidade de cadastros demorados e aplicativos pesados que dificultam a inclusão digital de todos os alunos.



💡 A Solução

O Videoaula App foi desenvolvido para simplificar esse processo. Ele resolve o problema da fragmentação no acesso ao ensino ao oferecer uma interface leve, direta e funcional, focada em produtividade acadêmica. Utilizando a infraestrutura do Jitsi Meet, o app permite a criação de salas seguras sem burocracia, garantindo que o foco permaneça no conteúdo da aula.



---



### 🚀 Tecnologias Utilizadas

*  **Linguagem:** Java s / Kotlin (Android)

*  **Videoconferência:** Jitsi Meet SDK (Open Source)

*  **IA Generativa:** Manus AI (Arquitetura de código e lógica de usabilidade)

*  **Interface:** Material Design com componentes customizados (HTML/CSS vibes aplicados ao Android)



---



### Instruções de instalação (passo a passo).

*  **Chamadas Criptografadas:** Integração nativa com salas do Jitsi.

*  **Interface Interativa:** Uso de botões de navegação estilizados e telas de "pré-fase" (lobby interativo).

*  **Assistência de IA:** Lógica de backend preparada para transcrição ou automação de notas de reunião.

*  **Dinâmica de Gamificação:** Implementação de barras de progresso e feedbacks visuais durante a interação.







---



### 🏗️ Arquitetura e Desenvolvimento

O desenvolvimento seguiu o fluxo de **Engenharia Reversa Assistida por IA**:

1. **Concepção:** Manus AI auxiliou na definição dos requisitos e na escolha do nicho.

2. **Modularização:** O código foi dividido em módulos (UI, Service, JitsiIntegration) para garantir escalabilidade.

3. **UI/UX:** Design focado em usabilidade móvel, com foco em botões de ação rápida e navegação intuitiva.



---



### 📂 Como rodar o projeto

1. Clone o repositório: https:https://github.com/victorsilvavsaraujo-create/videoaula-app

2. Abra no **Android Studio**.

3. Sincronize o Gradle (certifique-se de que a dependência do Jitsi Meet SDK está configurada).

4. Execute em um dispositivo real ou emulador com suporte a câmera/microfone.
